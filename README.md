# BLITZKRIEG

Tool for blitzkrieg method in Geometry Dash.

<img width="522" height="735" alt="image" src="https://github.com/user-attachments/assets/a75d8715-7726-461d-a2e6-d8e291ee69b3" />

## EN

**What is it**\n
A tool for the "blitzkrieg" method in GD. It generates runs starting from specific points (start positions).

**Build**
```
dotnet publish -c Release -o publish /p:SelfContained=true /p:PublishSingleFile=true
```
